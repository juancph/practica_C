# EJERCICIOS DE PRÁCTICA PARA EL LENGUAJE C
El siguiente repositorio, contiene una lista de ejercicios prácticos del lenguaje C. Comprende problemas de nivel inicial a intermedio, con el fin de poder aprender la sintáxis del lenguaje y las nociones básicas.

---
## Lista de ejercicios
1. [Acumulador](01_acumulador.c)
2. [Adivinar número](02_adivinar_numero.c)
3. [Verificador de número primo](03_encontrar_numeros_primos.c)
4. [Pirámide](04_piramide.c)
---
